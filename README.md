## Organisation des branches
dev : branche de développement.
Toutes les fonctionnalités sont développées à partir de cette branche.
Cette version de l’application n’est pas stable.

staging : branche de pré-production.
Elle sert à effectuer les derniers tests avant la fusion vers la branche main.
